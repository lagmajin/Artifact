# PROJECT GOALS

このアプリは、AfterEffectsのようなモーショングラフィックスを作成できる映像編集・合成ツールを目指します。

## 主な目標・ビジョン

- 直感的なレイヤー・タイムライン編集
- 高速なプレビューとリアルタイム再生
- 柔軟なエフェクト・トランジション・アニメーション
- ノードベース/レイヤーベース両対応の合成システム
- GPUアクセラレーションによる高品質レンダリング
- カスタムプラグイン・スクリプトによる拡張性
- プロフェッショナル向けのカラーグレーディング・オーディオ編集
- SNS/動画配信向けのテンプレート・プリセット
- プロジェクトの保存・復元・バージョン管理
- マルチプラットフォーム対応（Windows/Mac/Linux）

## ターゲットユーザー

- 映像クリエイター
- モーショングラフィックスデザイナー
- YouTuber・配信者
- 教育・研究用途の映像編集者

## 差別化ポイント

- 軽快な操作感と高速レンダリング
- オープンソースで拡張可能
- 既存のAfterEffectsプロジェクトとの互換性（将来的に）
- 独自のUI/UXによる新しい映像編集体験
