# Layer Creation Spec (2026-03-15)

## Goal
- レイヤー追加直後に編集へ入れる操作導線を標準化する。

## 1. 新規レイヤーは自動選択
- 対象: `addLayerToCurrentComposition(...)` で追加されたレイヤー。
- 挙動:
  - レイヤー追加成功直後に、その新規レイヤーを選択状態へ切り替える。
  - これにより直後に以下を行える状態にする。
    - 移動
    - 名前変更
    - 親設定

## 2. 新規レイヤーの位置（基本ルール）
- 基本ルール:
  - 追加時に選択レイヤーが存在する場合、新規レイヤーは選択レイヤーの直上（同じ表示グループ内で一つ上）へ配置する。
  - 選択レイヤーが存在しない場合は既存の追加位置（現実装の append）を維持する。
- 親の扱い:
  - 選択レイヤーが親を持つ場合、新規レイヤーはその親を継承する。
  - 選択レイヤーが親を持たない場合、新規レイヤーにも親を設定しない。

## Notes
- 本仕様は「新規追加直後の編集開始」を優先し、複雑な自動配置（時間位置・座標オフセット等）は対象外とする。
